Описание работы сохранения бд:
В каталоге /home/vlad/dump_db расположенны файлы main_db.py - (для создания дампа) и loader_db.py - (для вставки данных из файла).

Для работы этих файлов необходимы ключи .backup_key.pem.pub и .backup_key.pem ключи находятся в корневом каталоге пользователя vlad

Файл main_db.py срабатывает через crontab каждый понедельник в час дня. Этот файл создает дамп в папке file_dump вида backup_db-<datetime>.sql 

и создает зашифрованный архив файла в yandex object storage багет examquestions.

Файл loader_db.py работает по необходимости через команду python loader_db.py перед этим необходимо активировать виртуальное окружение с установленными 

библиотеками из requirements.txt. Файл скачивает последний файл дампа из yandex object storage разирхивирует и расшифровывает его. затем 

пушит данные в заранее установленную и настроенную бд. ВНИМАНИЕ! ПЕРЕД ТЕМ КАК ЗАПУШИТЬ ДАННЫЕ В БД ЭТОТ ФАЙЛ УДАЛЯЕТ ВСЕ ДАННЫЕ ИЗ БД. Все данные о бд

хранятся в переменных окружения файл .env.
